Creature Classes
The user controls the array of creatures using a variety of radio controls and buttons.

This project was for my IGME 102 class. It taught me how to use updateState and buttons to transition to different states. I worked with my peers to troubleshoot problems I was having.

Data Comparison
Data from a Monroe County census tract is visualized, and the user can click on two tracts to compare them.

This project was for my IGME 102 class. It was an earlier version of the Data Visualization project. In this version, mouseClicked is used to track where the user presses and finds the closest tracts. Completing this exercise was necessary to move on to the final result.

Data Visualization
Census data from Monroe County, NY, and Mecklenburg County, NC, are visualized and compared.

This project was for my IGME 102 class. The final result was built off a few previous exercises that introduced us to using JSON files. This project culminates what we learned during the semester about arrays, objects, menus, and functions.
